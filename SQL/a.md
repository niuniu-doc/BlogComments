
#### Order
##### 专车下单
```
1. 客服、会议宝、日租半日租子单、多日子单 不校验
2. 待支付订单校验
3. 非门童单3单校验
4. 10s冲突校验
```

##### 接送机下单
```
1. 机构、门童、客服下单不校验
2. 待支付订单(44)、行程中订单(15~44)(改派:6|3) 校验
3. 10s冲突校验
```

##### 巴士下单
```
1. 机构客服无限制
2. 行程中订单校验
```

##### 日租主单下单
```
1. 机构、客服、门童不校验
2. 行程中订单校验
3. 10s冲突校验
```

#### 多日主单下单
```
1. 机构、客服不校验
2. 行程中订单(3)校验
3. 10s冲突校验
```
